Analytics

One of the more recent projects I worked on was a scheduled job that downloads the Google Ad Reports for each of the hotels using our service and imported the data into our database. The data was then run through several algorithms to produce more usable bid analytics.

API Endpoints

Another recent project I worked on was creating several API endpoints to be used by a Angular.js front end application. I found this work to be greatly enjoyable and was a great opportunity to practice TDD.

Redis

One of the more complicated projects I built was combining the use of a Redis storage and a Resque job. The job would process hotels that needed to be notified of exceeding monthly budgets and if the budget was exceeded the ids would be stored in Redis and saved until other notification jobs have ran. Then the Redis set would be processed by a task to build up a notifications email list.

A Learning Project

I have been working on an application that allows a user to register clients via email. This email gets pushed into a queue to get processed by a task that gathers data based on the clients email address. An initial registration POST is sent that queues the client for a task that gets ran every day. A GET request can then be made to retrieve the recently produced client information. This is still a work in progress but the login and API are up and running. This has mostly been a side project to get more familiar with authentication for API’s as well as eventually working on an Angular.js app that will access this API.
